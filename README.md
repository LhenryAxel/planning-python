Membres du groupe :

- Axel Lhenry

Description du projet :
Ce programme permet de gérer un planning composé d’événements, de participants et de salles.
Les fonctionnalités principales sont :

Création d’un événement

Ajout d’un participant à un événement

Retrait d’un participant d’un événement

Affichage de l’agenda d’un participant

Recherche d’un créneau commun entre plusieurs participants

Suppression d’un événement

Gestion des salles (capacités, conflits de réservation)

Données :
Le programme utilise un fichier JSON nommé dico.json pour stocker toutes les données :

liste des événements

liste des participants

salles disponibles

Instructions d’installation et d’exécution :

Extraire le contenu du ZIP.

S’assurer que les fichiers suivants sont présents :

main.py

dico.json

README

Lancer le programme avec la commande :
python main.py

Compatibilité :
Python 3.8 ou supérieur.
